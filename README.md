# uniwind-plugin-next

> This is an unofficial plugin, and is not affiliated with Uniwind or Next.js in any way.

Next.js plugin for [Uniwind](https://uniwind.dev/) support. Note that only Webpack-based projects are supported, and there are no plans to support Turbopack. 

The implementation of this plugin is based on the official [Uniwind Vite plugin](https://docs.uniwind.dev/quickstart#vite), and aims to match it's functionality as closely as possible. See the limitations section below for known differences.
